# Tripleten web_project_around_auth

Este projeto é uma aplicação frontend que implementa registro, login, autenticação via token JWT, rotas protegidas e feedback de usuário com InfoTooltip, usando popups responsivos

Funcionalidades:

-Registro de usuário (email e senha)

-Login com persistência via localStorage

-Logout limpa token

-Rotas protegidas

-Popups responsivos para formulários e imagens

-InfoTooltip para mensagens de sucesso ou erro

Tecnologias:

HTML / CSS / JavaScript +

Fetch API

LocalStorage

Popups e Tooltips customizados
